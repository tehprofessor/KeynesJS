KeynesJS

Documentation

KeynesJS is an (attempt) at a simple javascript MVC with a focus on making it easier to develop and maintain web applications. Much of the framework's ideas are inspired by Ruby on Rails (because I think it [rails] is pretty smart).

Currently Keynes depends on jQuery and Mustache

KeynesianController

For managing and rendering different states

KeynesianModel

ORM supporting both browser and server based data storage

KeynesianView

Templates used to compose various states (powered by Mustache JS)
