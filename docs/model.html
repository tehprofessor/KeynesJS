Base(name, object) => Keynes.Model.Base

The base of Keynes.Model, contains most of the logic for the model Object. The object argument has a few reserved properties. All reserved properties are expected to be Objects themselves.

All properties which are not reserved, will be assigned to instances of the model.

Below are the reserved property names:

  • storage (remote, local)
  • has_many/belongs_to (association_name, associated_object)
  • attributes (property1, property2, ...)
Option Type Description
name String Name of the model (e.g. User)
object Object The definitions of the model.

Find(db, name) => Keynes.Model.Find

Logic pertaining to looking up data stored locally or remotely.
Option Type Description
db localStorage Key/value store provided by the browser
name String Name of model

Associations(instance, other_model_name, association_type) => [Keynes.Model.Base, ...]

Manages the logic for relationships between models.
Option Type Description
instance Object An `instance` of the model
other_model_name String The other model in the relationship
association_type String The type of association (e.g. belongs_to)
